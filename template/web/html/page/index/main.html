关于我们

南京金贝网络科技有限公司（KingBi Network Tech.）是国内领先的贵金属行业数据、信息和移动应用软件服务企业。

我们的核心团队均来自国内知名公司，公司提供全方位的移动客户端项目解决方案，秉着用户至上的理念，为客服提供贴心，优质的服务。

公司旗下自主研发的移动应用“口袋贵金属”基于移动互联网，为用户提供及时准确的贵金属交易信息。软件发布以来备受好评。

无论从界面还是体验上都得到了广大用户的一致认可。

目前，口袋贵金属应用已推出iPhone版，Android版，WindowsPhone版，WIN8版，WAP版，覆盖所有主流智能手机应用市场。
